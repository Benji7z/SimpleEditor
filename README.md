#SimpleEditor

## Functions
- Ability to save files
- Move Cursor with arrow keys
- The tab key (4 spaces)
- view file contemt
- Scrolling (via arrow keys)

## Wordwrap information
=> the wordwrap in the editor itself creates a new line therefore it might break some programs

## Purpose & Additional Information

This program was made to provide a lightweight texteditor with the most crucial functions such as the ones named above
therefore it is easy to overwrite a file by just exiting.
